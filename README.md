## 🚀 Project Overview
This is the initial setup for a web-based platform that supports:
- **Crowdfunding campaigns**
- **Clothes donation management**

Users will be able to:
- Create and manage campaigns
- Donate money or clothing
- Track donations and view histories

Admins will have full control over user accounts, campaigns, and donations.

## 🛠️ Planned Tech Stack

### Backend

### Database
- MySQL
